# UI-00 Preparar referencias y workflow Codex

> Status: prepared
> Scope: UI workflow organization

## Goal

Create a simple place to record UI implementation decisions without forcing Codex to load them on every task.

## Result

- UI task records live under `ai/workflows/ui/`.
- Workflow files use Notion task names.
- Records should stay short and describe final decisions, affected files and validation.

## Rule

Open these records only when the user asks about prior UI work or a related screen.
