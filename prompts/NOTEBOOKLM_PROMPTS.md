# NOTEBOOKLM_PROMPTS

Reusable prompts for NotebookLM over NeveraChef-AI docs.

## Product understanding

```text
Using the repository docs as source of truth, summarize NeveraChef-AI MVP scope, excluded scope, and open product questions in less than 12 bullet points.
```

## Architecture review

```text
Review ARCHITECTURE.md and DECISIONS.md. Identify the top 3 architectural risks for Sprint 2 and propose mitigations that do not add major complexity.
```

## UI and flow consistency

```text
Cross-check UI_SCREENS.md against PRODUCT.md and BACKLOG.md. List mismatches between confirmed MVP screens and future screens.
```

## AI integration readiness

```text
Based on AI_CONTEXT.md and BACKLOG.md, produce a Sprint 3 readiness checklist for AI text integration with clear go/no-go criteria.
```

## Notion sync support

```text
Read NOTION_SYNC.md and suggest an update order for Notion pages so product, architecture and backlog remain aligned with repo docs.
```
