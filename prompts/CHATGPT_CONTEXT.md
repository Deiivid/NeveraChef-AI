# CHATGPT_CONTEXT

Context pack for strategic/product/architecture discussions about NeveraChef-AI.

## What to assume

- Repo is KMP scaffold with shared Compose UI entrypoint.
- Product source-of-truth starts in Notion + Pencil and is synchronized into repo docs.
- MVP is Android-first delivery with shared UI strategy and local-first behavior.

## Ask ChatGPT for

1. Sprint-level decomposition with acceptance criteria.
2. ADR trade-off analysis.
3. Prompt quality improvements for recipe generation.
4. Risk assessment and mitigation planning.
5. Demo narrative and technical storytelling.

## Avoid asking without context

- Large refactors without reading docs.
- New feature requests not backed by Notion/Pencil/docs.
- Architecture rewrites that break MVP path.

- `docs/PROJECT_CONTEXT.md`
- `docs/PRODUCT.md`
- `docs/ARCHITECTURE.md`
- `docs/UI_SCREENS.md`
- `docs/BACKLOG.md`
- `docs/DECISIONS.md`
